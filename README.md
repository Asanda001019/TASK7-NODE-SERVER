 <h1> Node server </h1>
 <p>Task 11 creating a basic node.js server</p>

## Run Locally
Clone the project
```bash
  https://github.com/Asanda001019/Task11-NODE-SERVER.git
```
Go to the project directory
```bash
  cd weatherApp
```
Install dependencies
```bash
npm install

Start the react server
```bash
npm run dev

Start Node.js
nodemon server.js
```
